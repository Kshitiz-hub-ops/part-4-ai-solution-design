# AI Solution Design for Manufacturing Defect Detection

## Project Overview
This project proposes an AI-based solution for automated defect detection in manufacturing industries.

The objective is to reduce manual inspection effort and improve product quality using computer vision and deep learning.

---

## Selected Business Domain
Manufacturing

---

## Business Problem
Manual defect inspection is slow, costly, and prone to human errors. Products with scratches, dents, or stains may be missed during manual inspection.

---

## Proposed AI Solution
An AI-based computer vision system using a CNN model is proposed to automatically classify product images into:

- Normal
- Scratch
- Dent
- Stain

---

## Recommended Model
Convolutional Neural Network (CNN)
